How to Run

Because Chickenwing is a powerful security tool, macOS requires a few steps to run it safely:

  1. Download the .dmg and drag Chickenwing to your Applications folder.
  2. Right-click the app and select Open (this bypasses Gatekeeper for unsigned builds).
  3. The app will request Admin Privileges to access the network interface (en0/en1).
  4. Enjoy! No additional dependencies (like Python) are required.
